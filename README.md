<h2>Features</h2>
<li>API documentation generated by Swagger. <br></li>
<li>API allows PHP object communicate in Json Format. <br></li>
<li>Clients can get the products details. <br></li>
<li>Clients can create/edit/delete his/her users and he/she can get his/her users details.<br></li>
<li>JWT token system is implemented for authentication. </li>
<li>A refresh token system is implemented so that users do not need to re-login when token is expired</li>
<li>the API is RESTful and respect level 1, 2 and 3 of Richardson Maturity Model</li>

<h2>Prequisites</h2>
<li>PHP > 7.4.9<br></li>
<li>Symfony > 5.4 <br></li>
<li>Local server, e.g. XAMPP/WAMP for local use.<br></li>
<li>Libraries will be installed using Composer.<br></li>

<h2>Starting the application</h2>
You can run the application on your computer for development and testing purposes by following the simple steps below:<br>

<h3>Installation</h3>

<h4>Step1 Clone / Download</h4>
Clone the repository of this page.Open the folder "P7_api_rest_bilemo" 

<h4>Step 2 Configure environment variables</h4>
Open .env file and line 27 (database URL) with your own detail.

<h4>Step 3 Install all dependencies</h4>
Install Composer if you do not have it yet. </br>
In your cmd, go to the directory where you want to install the project and install dependencies with composer with the commands below:</br></br>

``` bash
$ cd some\directory
```

``` bash
$ composer install
```
All dependencies should be installed in a vendor directory.

<h4>Step 4 Create database</h4>
Create a new database using the command below:<br>

``` bash
$ php bin/console doctrine:database:create
```

``` bash
$ php bin/console doctrine:migrations:migrate
```
You can import fixture by running the command below:<br>
``` bash
$ php bin/console doctrine:fixtures:load
```

<h4>Generate the SSL keys</h4>

``` bash
$ php bin/console lexik:jwt:generate-keypair
```

Here you go! you can go to "/api" to see the API documentation !
